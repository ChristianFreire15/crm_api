Documentación

Está página está destinada para poner en conocimiento a los usuarios sobre el uso y configuración del API creada para el consumo de datos del sistema CRM. Mediante esta API podremos acceder a cualquier módulo que solicitemos información del CRM.

Para el consumo de datos mediante el API se utilizará el formato JSON con la siguiente estructura:

                    {
                        "module": "Clientes",
                        "fields": "id, name, nombre_completo_c, date_entered",
                        "limit": 10,
                        "relationship": {
                            "module": "Ventas",
                            "fields": "id, name, date_entered"
                        }
                    }

Module: Nombre del módulo a consultar (En el menú lateral en la opción "Módulos" podremos consultar los módulos habilitados para el consumo de datos).

Fields: Campos a consultar (En el menú lateral en la opción "Campos" podremos consultar los campos habilitados según el módulo requerido para el consumo de datos).

Limit: Límite de registros a consultar

Relationship: Consultar un módulo relacionado con el módulo principal a consultar opción "Module".

Relationship -> Module: Nombre del módulo relacionado a consultar.

Relationship -> Fields: Campos del módulo relacionado a consultar.

Todas las opciones son requeridas.
